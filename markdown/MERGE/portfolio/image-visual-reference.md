## ğŸ“ Key Takeaways

1. **Wrapper Pattern**
   - Frame (wrapper) = aspect ratio container
   - Image (content) = fills container

2. **Essential Properties**
   - Frame: `position: relative`, `aspect-ratio`, `overflow: hidden`
   - Image: `position: absolute`, `object-fit: cover`

3. **Responsive Strategy**
   - Start with natural aspect ratio
   - Adjust at breakpoints based on screen shape
   - Portrait â†’ Landscape on mobile (usually)

4. **Common Mistakes**
   - Don't put `object-fit` on wrapper
   - Don't forget `position: relative` on frame
   - Don't skip `overflow: hidden`
